Bandpass-Filter

For filtering the digital signals a butterworth filter with the order 10 has been used. It has been designed as a bandpass with the passing frequency. 

Window

For a better processing with the FFT of the signal, a Windows has to be applyed. In this case a Hanning Window has been choosen.
